# Graph RAG Travel System

Travel recommendation system using Graph RAG for enriched place suggestions.

## Setup

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
pip install -r requirements.txt
python server.py
```

## Stack

**Frontend**: React, Bootstrap, Google Maps API
**Backend**: Flask, NetworkX, ArangoDB
